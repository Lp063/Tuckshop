
# Tuckshop

## Installation

[Get Docker](https://docs.docker.com/get-docker/)

Once installed Run the following command:

### `cd Tuckshop`

### `docker-compose up --build`

`import db/tuckshop.sql into Tuckshop database in phpmyadmin`

## servers

### API
`http://localhost:4000`

### phpmyadmin
`http://localhost:8080`

### Frontend
`http://localhost:3000`
